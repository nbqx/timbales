# timbales

timbre + ls => timbales.

slight tools for playing loop music with [LiveScript](https://github.com/gkz/LiveScript) and [timbre](https://github.com/mohayonao/timbre).

### Install

`$ (sudo) npm install -g LiveScript`

then

`$ slake build`
